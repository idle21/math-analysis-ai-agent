# math-analysis-ai-agent# 数学分析智能学习助手
本项目是数学分析大作业——「AI工具在数学学习中的应用」的实现，基于 Python + Flask 搭建了简易智能体，支持数学分析核心知识点（极限、导数、积分、级数）的问答、例题解析、习题推荐功能。

## 功能说明
1. 知识点问答：通过关键词匹配返回极限/导数/积分/级数的核心定义、性质；
2. 例题解析：针对核心知识点提供典型例题及详细推导过程；
3. 习题推荐：配套知识点的练习题，辅助巩固学习；
4. 公式渲染：基于 MathJax 支持 LaTeX 数学公式实时展示。
